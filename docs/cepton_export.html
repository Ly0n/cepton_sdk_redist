Cepton Export¶

The Cepton Export tool allows exporting LiDAR points in various file formats.

Install¶

Requirements:

  • Python 3

Install the Cepton Python SDK

pip3 install --user -U cepton_sdk[samples]

For usage, run

cepton_export.py -h
